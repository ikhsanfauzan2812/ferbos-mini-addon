API Documentation

Internal Endpoints (No Authentication Required)

GET /api - API information
GET /status - Addon status
GET /ping - Health check
GET /tables - List database tables
GET /entities - List entities
GET /states - Get states
POST /query - Execute SQL query

External Endpoints (Authentication Required)

GET /external/status - External status
POST /external/query - External query
GET /external/entities - External entities
GET /external/states - External states

WebSocket Endpoints

/ws - WebSocket connection for real-time updates
Events: - connect: Connection established - disconnect: Connection closed - query_database: Execute SQL query - subscribe_entity: Subscribe to entity updates - database_updated: Database change notification

Authentication

External endpoints require authentication via API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Rate Limiting

External endpoints are rate limited to prevent abuse. Default limit is 100 requests per minute per IP.
